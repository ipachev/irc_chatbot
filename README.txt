# Lab 4 - Chatbot
Skylar Durst, Ivan Pachev, Jeremy Kerfs


# Requirements

- Python 3.0
- NLTK 3.0
- IRC 15.0

# Launching the Chatbot

```
python bot.py <IRC_URL> <IRC_CHANNEL> <BOT_NAME>
```

Note: The IRC Channel should be surrounded by quotes (i.e. "#CPE582").

One IRC server that is known to work is `irc.freenode.net`
